RXB Core
=============

Setup
---------------------
RXB is an independent cryptocurrency using proven blockchain technology and cryptographic libraries from Bitcoin's architecture. It has its own blockchain starting from a custom genesis block.

**Key specs:**
- P2P Port: 8327
- RPC Port: 8328
- No Ordinals/Inscriptions

**Downloads:**
- Binaries: [rxbcore.org](https://rxbcore.org)
- Source: [github.com/Heiwabitnull/rxb-core](https://github.com/Heiwabitnull/rxb-core)
- Whitepaper: [Read here](https://raw.githubusercontent.com/Heiwabitnull/RXBWhitepaper/refs/heads/main/RXBWhitepaper_EN.md)

Running
---------------------

### Linux

Unpack the files into a directory and run:
- `bin/rxb-qt` (GUI) or
- `bin/rxbd` (headless)

### Windows

Unpack the files into a directory, and then run rxb-qt.exe.

### macOS

**Note:** macOS version is not yet tested. Proceed at your own risk.

Drag RXB Core to your applications folder and then run RXB Core.

**Website:** [rxbcore.org](https://rxbcore.org)
